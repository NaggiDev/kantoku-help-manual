# プロジェクトコスト

効果的な財務監督のためにプロジェクト費用、予算配分、コスト管理を追跡します。

## コスト管理

![プロジェクトコスト](/guide-books/web-version/08-project-cost.jpg)

プロジェクトコスト機能には以下が含まれます：
- プロジェクト予算と費用の監視
- プロジェクトフェーズごとのコスト追跡
- コストレポートと分析の生成
- 予算配分の管理
- リソースコストの監視
- 実際と計画費用の比較
