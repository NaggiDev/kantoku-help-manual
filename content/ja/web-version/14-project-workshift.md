# プロジェクトワークシフト

最適なカバレッジと生産性を確保するためにプロジェクトチームの勤務シフトを整理・管理します。

## ワークシフト管理

![プロジェクトワークシフト](/guide-books/web-version/14-project-workshift.jpg)

ワークシフト管理機能：
- 勤務シフトの作成と割り当て
- シフトローテーションとパターンの管理
- シフトカバレッジと可用性の追跡
- シフト変更リクエストの処理
- シフトベースの生産性監視
- シフトレポートと分析の生成
