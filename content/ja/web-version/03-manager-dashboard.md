# マネージャーダッシュボード

マネージャーダッシュボードは、チーム、プロジェクト、組織運営を監督するための包括的なツールを提供します。

## マネージャーダッシュボード機能

![マネージャーダッシュボード](/guide-books/web-version/03-manager-dashboard.jpg)

マネージャーとして、以下のことができます：
- チームのパフォーマンスと勤怠を監視
- 複数のプロジェクトを同時に監督
- 高度なレポートと分析にアクセス
- 従業員のスケジュールと割り当てを管理
- リクエストの確認と承認
- プロジェクトの進捗と締切を追跡
