# マネージャーダッシュボード（モバイルアプリ）

モバイルマネージャーダッシュボードでどこからでも重要な管理ツールにアクセスできます。

## モバイルマネージャー機能

![マネージャーダッシュボードアプリ](/guide-books/app-version/02-manager-dashboard.jpg)

モバイルマネージャーダッシュボードでは以下が提供されます：
- リアルタイムチーム監視
- 迅速な承認ワークフロー
- モバイル最適化レポート
- 緊急通知システム
- 位置ベースのチーム追跡
- 簡素化された管理コントロール
